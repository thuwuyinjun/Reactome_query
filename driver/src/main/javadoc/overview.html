This is the reference driver implementation for Neo4j, it allows you to connect to a Neo4j server and interact with it. If you are using Java, this is generally the interface you'd use to integrate Neo4j in your application.

Getting Connected

Once you have the driver and a connector on your classpath, you can establish sessions with Neo4j.

Driver driver = GraphDatabase.driver( "bolt://localhost", AuthTokens.basic( "neo4j", "p4ssw0rd" ) );
try( Session session = driver.session() )
{
    StatementResult result = session.run( "RETURN 'hello, world!'" );
    while( result.hasNext() )
    {
        Record record = result.next();
        System.out.println( record.get(0).asString() );
    }
}
Have a look at the Driverclass to get started.
